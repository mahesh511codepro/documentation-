ITSCORE – Documentation

Created by Themelevate

1. Overview

IT CORE is a modern, dark-themed multipurpose IT service HTML template. It includes 10 professional categories such as Computer Repair, Software Company, Web Hosting, Cybersecurity, Mobile App Developer, SEO Agency, and more.

2. Folder Structure

ITCORE/
│
├── 01-computer-repair/
├── 02-software-company/
├── 03-managed-it-services/
├── 04-cybersecurity-consultant/
├── 05-computer-support/
├── 06-web-hosting/
├── 07-network-security/
├── 08-mobile-app-developer/
├── 09-website-designer/
├── 10-seo-agency/
└── documentation/

3. How to Edit

Edit any HTML file using your preferred editor (VS Code recommended).

Common Files:

/assets/css/style.css – Main CSS file
/assets/js/main.js – Main JS file
/assets/images/ – Template images

4. How to Change Logo

Replace the logo inside:
